# Проект Data Processing
Описание проекта
Проект предназначен для обработки списков финансовых операций. Основные цели включают фильтрацию операций по статусу и сортировку операций по дате.

Тестирование
Проект оснащен автоматизированными тестами, которые обеспечивают контролькачества и устойчивость программного продукта. Тесты написаны с использованием фреймворка pytest и регулярно выполняются для проверки работоспособности ключевых компонентов.

Структура тестов
Тесты организованы в папке /tests. Основное внимание уделено следующим направлениям:

Unit-теста: проверка изолированного поведения отдельных функций и классов. Интеграционное тестирование: проверка взаимодействия компонентов друг с другом. Тестирование производительности: (при наличии бенчмарков) оценка скорости выполнения операций.
