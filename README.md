### Тестирование. Pytest

# Основные функциональные возможности
Фильтрация транзакций по состояниям (например, выполненные или отменённые операции).
Сортировка транзакций по дате исполнения.
Маски конфиденциальности, позволяющие скрывать личные данные пользователей.
Расширенное тестирование с покрытием более 80%, гарантирующее стабильность и надёжность.

# Использование
Запуск основной логики возможен через консоль: python -m src.processings
pytest

# Структура проекта
Проекты организован следующим образом:

financial-tracker/
├── src/
│   ├── processings.py  # Логика фильтрации и сортировки транзакций
│   ├── widget.py        # Вспомогательные утилиты для работы с данными
│   └── masks.py        # Методы маскирования конфиденциальной информации
├── tests/              # Директория с юнит-тестами
│   ├── test_processings.py
│   ├── test_widget.py
│   └── test_masks.py
├── .gitignore          # Игнорируемые файлы
├── README.md           # Текущий документ
└── requirements.txt    # Список зависимостей

# Установка
Склонируйте репозиторий и установите зависимости:

git clone git@github.com:<your_username>/financial-tracker.git
cd financial-tracker
python3 -m venv env
source env/bin/activate  # Linux/MacOS
env\Scripts\activate    # Windows
pip install -r requirements.txt
