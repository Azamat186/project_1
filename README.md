### Homework 11.1
## Модуль Generators
Модуль 'Generators' содержит функции, предназначенные для удобной обработки финансовых транзакций и генерации номеров 
банковских карт.

## Функции 

- **`filter_by_currency(transactions, currency)`**: Фильтрация транзакций по конкретной валюте.
- - **`transaction_descriptions(transactions)`**: Получение описаний транзакций.
- **`card_number_generator(start=1, stop=9999999999999999)`**: Генерация последовательных номеров банковских карт.
